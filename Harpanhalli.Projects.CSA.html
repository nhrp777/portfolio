Projects - AP CSA

Android Studio Mobile Application, CSA

December 8 - 15, 2017

Project Objective: Design, build, and test an Android mobile application using Android Studio.

In this project, we were asked to use Android Studio to develop a mobile application. I worked with 3 other students to develop the OrElse mobile application, which allows a student to store and prioritize upcoming homework assignments for each of their classes. The student will manually input each assignment, giving it a due date and priority multiplier, as well as inputting their current grade in the class. The assignments will then be displayed, and can be organized by priority multiplier, due date, or class grade. This application was designed and developed as a proof of concept -- all information was local to the device.

In my role, I designed and programmed the application's login screen, and documented the main part of the project. I also worked on designing the prototype in Proto.io, creating the Assignment List, and programming the add and remove assignment functions. This project helped me learn the fundamentals of app development in Android Studio; more specifically, user input, app navigation, and how to write JUnit test classes. We identified a problem in our community and learned how to logically isolate each component to address and solve it. I approached the problem through user-centric design, exploring how the user will interact with each component of our application.
